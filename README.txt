This plugin allows you to add Fib as payment gateway for your woocommerce store
with easy installation

== Installation ==

Using Wordpress Admin Panel:
1. Download the project as .zip file from Code -> Download Zip
2. Go to Admin Panel -> Plugins -> Add New and select Upload Plugin
3. Select `wp-fib-gateway.zip` and then click on upload.
4. Activate the plugin
5. Add your secret key from WooCommerce -> Settings -> Payments -> Fib Gateway
6. Thats it :)

Using File Manager:
1. Download the project as .zip file from Code -> Download Zip
2. Upload `wp-fib-gateway.zip` to the `/wp-content/plugins/` directory
3. Extract wp-fib-gateway.zip
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Add your secret key from WooCommerce -> Settings -> Payments -> Fib Gateway
6. Thats it :)
